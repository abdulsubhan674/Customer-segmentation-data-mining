# Customer Segmentation & Data Mining Project

##  Overview
This project applies multiple data mining techniques to analyze customer purchasing behavior and perform segmentation.

##  Key Features
- Data Cleaning & Preprocessing
- Feature Engineering (RFM Analysis)
- Clustering:
  - K-Means
  - DBSCAN
- Dimensionality Reduction (PCA)
- Association Rule Mining:
  - Apriori
  - FP-Growth
- Classification Models:
  - Naive Bayes
  - Support Vector Machine (SVM)

##  Techniques Used
- Clustering & Unsupervised Learning
- Market Basket Analysis
- Classification & Prediction
- Model Evaluation (Accuracy, ROC, Silhouette Score)

##  Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- MLxtend
- Matplotlib, Seaborn

##  Outcome
- Identified customer segments
- Extracted purchasing patterns
- Built predictive models for classification

##  Dataset
Online retail dataset 
 https://www.kaggle.com/datasets/carrie1/ecommerce-data
